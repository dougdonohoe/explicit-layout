System Requirements

ExplictLayout can be used on any platform that supports Java 2 SDK.

Installation Instructions

To install ExplicitLayout, extract the zip file explicit_version.zip provided, into a directory of your choice.

Running the ExplicitLayout Demonstration

The ExplicitLayout Demonstration is an application containing interactive layout examples complete with source code. The application is provided for the purposes of evaluating and learning how to use ExplicitLayout; it is available in all editions.

There are two ways to start the application:

1. If your platform supports executable jar files, double click on demo.jar

2. If step 1 did not do anything, change to the directory containing demo.jar at a command prompt and type:

java -jar demo.jar

When the application frame is showing, follow the on screen instructions.

Developing Applications

To use ExplicitLayout in your own applications, include one of the ExplicitLayout jar files in your classpath:

Description JAR file Packages
Minimum elcore.jar com.zookitec.layout
Recommended explicit.jar com.zookitec.layout
com.zookitec.layout.styles
All elall.jar com.zookitec.layout
com.zookitec.layout.styles
com.zookitec.layout.source
