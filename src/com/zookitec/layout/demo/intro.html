ExplicitLayout v3.0 Demonstration

Create sophisticated, adaptable and professional user interfaces. ExplicitLayout is the ultimate layout manager for Java with a unique approach to defining resolution independent layouts for AWT, Swing or JavaBean components. ExplicitLayout gives the developer complete control over each component's resizing behaviour as the container size or application state changes. ExplicitLayout extends the JDK with a powerful, flexible, intuitive and extensible layout manager that facilitates the implementation of almost any component layout.

This application demonstrates several example component layouts implemented a using ExplicitLayout 3.0 Standard and Professional.

To view an example, please select one from the Standard or Professional menu.
You can resize each example frame, using your mouse, to see how it affects the layout.
You can view the source code to see how each example is implemented.
